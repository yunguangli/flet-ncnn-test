# Suggested Commands

| Action | Command |
|---|---|
| Run web app (dev) | `uv run flet run --web` |
| Run desktop app | `uv run flet run` |
| Run tests | `uv run pytest` or `uv run flet test` |
| List deps | `uv pip list` |
| Add dep | `uv add <package>` |
| Build APK | `flet build apk -v` |
| Build web | `flet build web -v` |
| Git status | `git status` |
| Git log | `git log --oneline -10` |
