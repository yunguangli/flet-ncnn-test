# Suggested Commands

| Action | Command |
|---|---|
| Run web (primary dev) | `uv run flet run --web` |
| Run desktop (OpenCV camera) | `uv run flet run` |
| Tests | `uv run pytest` or `uv run flet test` |
| Add dependency | `uv add <package>` |
| Build APK / web | `flet build apk -v` / `flet build web -v` |
| Serena memory check | `serena memories check` |
